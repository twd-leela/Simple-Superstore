# Simple Super Store Dashboard
# Objective :
 To design a simple super store dashboard in Tableau Desktop that helps visualize key business metrics such as Total sales, Total profit, orders, Discount, Region - wise sales, category - wise performance, and Top - 5 customers using the sample super store dataset.
## Key Insights section 
Identified high - performing regions and categories.
Noticed that selective customers contribute a large share of total sales.
Saw that certain categories generate high sales but lower profit, suggesting pricing or cost - optimization opportunities.
